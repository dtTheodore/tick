# Tick UI — Off-Chain Core Tap UX (v0) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship `games/tap-trading/ui/` — a single-screen React 19 + Vite + TS surface that lets a tester see a live ETH price grid with per-cell multipliers, tap a cell, watch it settle to W/L/V against the existing off-chain backend stack (api + worker + aggregator).

**Architecture:** New project crate-equivalent under `games/tap-trading/ui/`, mirroring `games/2048/ui/` layout. React 19 with no global state library — `@tanstack/react-query` owns server cache, `useSyncExternalStore` + a module-scoped pub/sub owns the 20 Hz oracle tick stream. The Rust `tap-trading-pricing-engine` is hand-ported to TypeScript at `src/pricing/`; a new Rust fixture-generator binary emits a parity JSON the TS port asserts against in `bun test`. Wall-clock-aligned cell grid + per-cell multiplier compute selectors keep React re-renders selective. Identity is a localStorage UUID injected as the `X-Account-Id` header; the backend's existing dev middleware lazy-creates the account.

**Tech Stack:** React 19, Vite 7, TypeScript (strict), Tailwind 4 + shadcn/ui (Radix primitives), Bun (runtime + package manager + test runner), `@tanstack/react-query` 5, `tokio-tungstenite`-style WebSocket via browser `WebSocket`. New Rust binary in `games/tap-trading/backend/pricing-engine/src/bin/gen_fixtures.rs` for parity fixtures. No Jotai, no Zustand, no Redux, no Zod, no Vitest (bun test runs the TS parity).

**Spec:**
- `docs/superpowers/specs/2026-05-28-tick-ui-off-chain-design.md` — the canonical design this plan implements.
- `games/tap-trading/docs/PRD.md` §3 (palette, fonts), §8 (MVP scope), §10 (MVP-02..MVP-08, MVP-10).
- `docs/decisions/0009-tick-api-cross-service-contracts.md` — POST /v1/positions shape + error taxonomy.
- `games/tap-trading/docs/MATH_SPEC.md` §4 — multiplier formula the TS port must match.
- `games/tap-trading/backend/pricing-engine/src/` — the canonical Rust impl to mirror.
- `games/2048/ui/` — the FE convention this plan copies (vite.config.ts shape, tsconfig, components.json).
- `platform/backend/api/gateway/src/lib.rs` — the `CorsLayer` pattern already adopted by the tap-api.

**Backend assumptions (frozen at `78c9a43`+):**
- `tap-trading-api` reads `TAP_DB_URL` / `TAP_REDIS_URL` / `TAP_AGGREGATOR_URL` / `TAP_API_PORT` env vars.
- `CorsLayer::permissive()` is on (commit `536b37c`).
- API serves: `GET /healthz`, `GET /metrics`, WS `GET /stream`, `GET /v1/ping`, `GET /v1/me`, `GET /v1/me/history`, `POST /v1/positions`, `GET /v1/positions/:id`, all `/v1/*` behind `X-Account-Id` middleware.
- Stake tiers `{50, 100, 500, 1000}`; v0 fixes stake at **100**.
- All endpoints accept `X-Account-Id: <any-string-up-to-128-chars>` and lazy-create the account with a 10k signup bonus.
- Cell timing: `t_close_ms - t_open_ms === 5000`, `t_open_ms % 5000 === 0`, tap blocked when `now + 1000 >= t_close_ms`.
- Per-account rate limit: 10 req/s burst 10.
- Idempotency: dedup on `(account_id, client_request_id)` UUID.

```ts
export function getAccountId(): string {
  const existing = localStorage.getItem(KEY);
  if (existing) return existing;

  localStorage.setItem(KEY, fresh);
  return fresh;
}

export function resetAccountId(): void {
  localStorage.removeItem(KEY);
}
```

```ts
import { env } from './env';
import { getAccountId } from './identity';

export class ApiError extends Error {
  constructor(
    public status: number,
    public code: string,
    public body?: unknown,
    public retryAfterSecs?: number,
  ) {
    super(`api ${status} ${code}`);
  }

  /** Server's recomputed μ from the drift_exceeded error body, if present. */
  get serverMultiplier(): number | undefined {
    if (this.body && typeof this.body === 'object' && 'server_multiplier' in this.body) {
      const v = (this.body as { server_multiplier: unknown }).server_multiplier;
      return typeof v === 'number' ? v : undefined;
    }
    return undefined;
  }
}

interface RequestOptions {
  method?: 'GET' | 'POST';
  body?: unknown;
  signal?: AbortSignal;
}

export async function api<T>(path: string, opts: RequestOptions = {}): Promise<T> {
  const res = await fetch(`${env.apiUrl}${path}`, {
    method: opts.method ?? 'GET',
    headers: {
      'X-Account-Id': getAccountId(),
      ...(opts.body ? { 'Content-Type': 'application/json' } : {}),
    },
    body: opts.body ? JSON.stringify(opts.body) : undefined,
    signal: opts.signal,
  });
  if (!res.ok) {
    let body: unknown = undefined;
    let code = String(res.status);
    try {
      body = await res.json();
      if (body && typeof body === 'object' && 'error' in body && typeof body.error === 'string') {
        code = body.error;
      }
    } catch { /* ignore parse failure */ }
    const retryHeader = res.headers.get('retry-after');
    const retryAfterSecs = retryHeader ? Number(retryHeader) : undefined;
    throw new ApiError(res.status, code, body, Number.isFinite(retryAfterSecs) ? retryAfterSecs : undefined);
  }
  return (await res.json()) as T;
}
```

```ts
import { env } from './env';
import { tickStore, type OracleTick } from './tick-store';

let socket: WebSocket | null = null;
let reconnectAttempts = 0;
let reconnectTimer: ReturnType<typeof setTimeout> | null = null;

function backoffMs(): number {
  return Math.min(4000, 250 * 2 ** reconnectAttempts);
}

function scheduleReconnect() {
  if (reconnectTimer) return;
  tickStore.setWsState('reconnecting');
  reconnectTimer = setTimeout(() => {
    reconnectTimer = null;
    reconnectAttempts += 1;
    connect();
  }, backoffMs());
}

function parseTick(msg: unknown): OracleTick | null {
  if (!msg || typeof msg !== 'object') return null;
  const m = msg as Record<string, unknown>;
  if (m.type !== 'tick') return null;
  if (typeof m.asset !== 'string'
      || typeof m.run_id !== 'number'
      || typeof m.seq !== 'number'
      || typeof m.ts_ms !== 'number'
      || typeof m.mid !== 'number'
      || typeof m.vol_annualized !== 'number'
      || typeof m.source_count !== 'number') {
    return null;
  }
  if (m.asset !== 'ETH' && m.asset !== 'BTC' && m.asset !== 'SOL') return null;
  return {
    asset: m.asset,
    run_id: m.run_id,
    seq: m.seq,
    ts_ms: m.ts_ms,
    mid: m.mid,
    vol_annualized: m.vol_annualized,
    source_count: m.source_count,
  };
}

export function connect() {
  if (socket && (socket.readyState === WebSocket.OPEN || socket.readyState === WebSocket.CONNECTING)) {
    return;
  }
  tickStore.setWsState('connecting');
  socket = new WebSocket(env.wsUrl);
  socket.onopen = () => { reconnectAttempts = 0; };
  socket.onmessage = (ev) => {
    try {
      const msg = JSON.parse(ev.data as string);
      const tick = parseTick(msg);
      if (tick) tickStore.push(tick);
      // status and heartbeat frames are observed but not consumed in v0.
    } catch {
      // drop garbled frames
    }
  };
  socket.onclose = () => { scheduleReconnect(); };
  socket.onerror = () => { socket?.close(); };
}

export function disconnect() {
  if (reconnectTimer) clearTimeout(reconnectTimer);
  reconnectTimer = null;
  socket?.close();
  socket = null;
  tickStore.setWsState('closed');
}
```

```ts
/**
 * Complementary error function. Abramowitz & Stegun 7.1.26.
 * Max abs error: 1.5e-7 (vs libm erfc). Sufficient for our 1e-6 parity tolerance.
 */
export function erfc(x: number): number {
  const z = Math.abs(x);
  const t = 1 / (1 + 0.5 * z);
  const ans = t * Math.exp(
    -z * z - 1.26551223 +
    t * (1.00002368 +
    t * (0.37409196 +
    t * (0.09678418 +
    t * (-0.18628806 +
    t * (0.27886807 +
    t * (-1.13520398 +
    t * (1.48851587 +
    t * (-0.82215223 +
    t * 0.17087277))))))))
  );
  return x >= 0 ? ans : 2 - ans;
}
```

## Task 7: Pricing port — volatility (EWMA)

**Files:**
- Create: `games/tap-trading/ui/src/pricing/vol.ts`
- Test: `games/tap-trading/ui/src/pricing/__tests__/vol.test.ts`

This port mirrors **two** Rust functions from `pricing-engine/src/vol.rs`:

1. `estimate_realized_vol(log_returns: &[f64], lambda: f64) -> Result<f64, PricingError>` — EWMA over a slice of log returns, seeded with `r[0]²`, annualized via `√SECONDS_PER_YEAR`. Returns errors for empty input (`InsufficientHistory`), λ outside `[0, 1)` (`InvalidLambda`), or any non-finite return (`InvalidLogReturn`).
2. `jump_adjusted_sigma(raw_sigma_annualized, last_log_return, prev_sigma_annualized) -> (f64, bool)` — post-EWMA spike absorption: when `|r| > 5 · prev_per_sec`, bump σ̂ to `|r| · √yearly`, taking the max with the raw σ̂ (never downgrade). NaN raw propagates.

We express both as a single pure `nextVol(prev, newLogReturn, lambda)` state transition for streaming use — but the *math* and *error semantics* are identical to the Rust originals.


Specifically note:
- Variance is seeded with `r[0]²` on the first observation (no zero start). With a single observation, `σ_annualized = |r| · √SECONDS_PER_YEAR`.
- λ outside `[0, 1)` → `InvalidLambda`. `NaN` λ also → `InvalidLambda`.
- Non-finite return → `InvalidLogReturn`. The previous state is **not** silently preserved.
- Spike branch: `|r| > 5 · (prev_annual / √yearly)` triggers, otherwise pass through. The bumped value is `max(raw, |r| · √yearly)`.
- NaN raw propagates through both branches.

- [ ] **Step 4: Implement `vol.ts`**

**Critical parity invariant.** Rust's `estimate_realized_vol` accumulates a *raw* variance over the input slice; `jump_adjusted_sigma` is a separate pure post-step that may bump the *displayed* σ but does NOT feed back into the EWMA accumulator. The streaming port must keep these separate: the next EWMA step reads the **raw** prior variance, not the (potentially bumped) σ. If you collapse them, a single spike permanently elevates the EWMA — a slow-creeping divergence from Rust that the multiplier parity test does NOT catch (it takes σ as input).

```ts
import { SECONDS_PER_YEAR } from './constants';

export class InvalidLambda extends Error {
  constructor(public lambda: number) { super(`invalid lambda: ${lambda}`); }
}
export class InvalidLogReturn extends Error {
  constructor(public value: number) { super(`invalid log return: ${value}`); }
}
/** Mirrors PricingError::InsufficientHistory — empty log-return slice. */
export class InsufficientHistory extends Error {
  constructor() { super('insufficient history'); }
}

export interface VolState {
  /** Accumulator — feeds the next EWMA step. Bumping does NOT touch this. */
  rawVarianceTickSq: number;
  /** What consumers read; may be bumped above raw on spikes. */
  sigmaAnnualized: number;
  lastLogReturn: number | null;
}

const YEARLY_ROOT = Math.sqrt(SECONDS_PER_YEAR);

/**
 * Streaming-style port of `tap_trading_pricing_engine::vol`. Two-step:
 *   1. EWMA on raw variance:
 *        cold-start (lastLogReturn === null): rawVar := r²
 *        steady:                                rawVar := λ·rawVar_prev + (1 − λ)·r²
 *      `rawAnnual = √rawVar · √SECONDS_PER_YEAR`.
 *   2. jump_adjusted_sigma: if `|r| > 5 · (prev_σ / √yearly)`, bump the
 *      displayed σ to `max(rawAnnual, |r| · √yearly)`. The raw variance is
 *      NOT bumped — that's the parity-critical detail.
 *
 * Throws on bad inputs (mirrors PricingError::InvalidLambda / InvalidLogReturn).
 */
export function nextVol(prev: VolState, newLogReturn: number, lambda: number): VolState {
  if (!(lambda >= 0 && lambda < 1)) throw new InvalidLambda(lambda);
  if (!Number.isFinite(newLogReturn)) throw new InvalidLogReturn(newLogReturn);

  const r2 = newLogReturn * newLogReturn;
  const rawVarianceTickSq = prev.lastLogReturn === null
    ? r2
    : lambda * prev.rawVarianceTickSq + (1 - lambda) * r2;

  const rawAnnual = Math.sqrt(Math.max(0, rawVarianceTickSq)) * YEARLY_ROOT;

  // jump_adjusted_sigma — read prev displayed σ for the spike threshold,
  // emit max(raw, bumped) for the displayed value. Does NOT touch rawVariance.
  const isSpike = prev.sigmaAnnualized > 0
    && Math.abs(newLogReturn) > 5 * (prev.sigmaAnnualized / YEARLY_ROOT);
  const bumped = isSpike ? Math.abs(newLogReturn) * YEARLY_ROOT : 0;
  const sigmaAnnualized = Math.max(rawAnnual, bumped);

  return { rawVarianceTickSq, sigmaAnnualized, lastLogReturn: newLogReturn };
}

export const INITIAL_VOL_STATE: VolState = {
  rawVarianceTickSq: 0,
  sigmaAnnualized: 0,
  lastLogReturn: null,
};
```

```ts
export class InvalidTerms extends Error {
  constructor() { super('hui_no_touch: terms must be > 0'); }
}
export class InvalidBand extends Error {
  constructor(public l: number, public h: number) {
    super(`hui_no_touch: invalid band l=${l} h=${h}`);
  }
}
export class HuiConvergenceFailure extends Error {
  constructor(public lastTermMag: number, public terms: number) {
    super(`hui series failed to converge: last_term_mag=${lastTermMag} terms=${terms}`);
  }
}

const CONVERGENCE_TOLERANCE = 1e-4;

export function huiNoTouch(
  s0: number, l: number, h: number, sigmaPerSec: number, tauSec: number, terms: number,
): number {
  if (terms === 0) throw new InvalidTerms();
  if (!(l > 0 && h > l
        && Number.isFinite(s0) && Number.isFinite(sigmaPerSec) && Number.isFinite(tauSec))) {
    throw new InvalidBand(l, h);
  }
  if (tauSec <= 0) return 1.0;          // degenerate window — Rust returns Ok(1.0)
  if (s0 <= l || s0 >= h) return 0.0;   // already touched — Rust returns Ok(0.0)

  const alpha = 0.5;
  const beta = -0.25;
  const z = Math.log(h / l);
  const logS0OverL = Math.log(s0 / l);
  const s0OverLAlpha = Math.pow(s0 / l, alpha);
  const s0OverHAlpha = Math.pow(s0 / h, alpha);

  let sum = 0;
  let lastTermMag = Infinity;

  for (let n = 1; n <= terms; n++) {
    const piNOverZ = (Math.PI * n) / z;
    const sign = n % 2 === 0 ? 1 : -1;
    const numerator = ((2 * Math.PI * n) / (z * z)) * (s0OverLAlpha - sign * s0OverHAlpha);
    const denominator = alpha * alpha + piNOverZ * piNOverZ;
    const sinTerm = Math.sin(piNOverZ * logS0OverL);
    const expArg = -0.5 * (piNOverZ * piNOverZ - beta) * sigmaPerSec * sigmaPerSec * tauSec;
    const expTerm = Math.exp(expArg);

    const term = (numerator / denominator) * sinTerm * expTerm;
    sum += term;
    lastTermMag = Math.abs(term);

    if (lastTermMag < CONVERGENCE_TOLERANCE && n >= 3) {
      return Math.max(0, Math.min(1, sum));
    }
  }

  throw new HuiConvergenceFailure(lastTermMag, terms);
}
```

5. `raw = p_touch < EPSILON ? multiplier_cap : (1.0 − house_margin) / p_touch`.
6. `Ok(raw.max(floor).min(multiplier_cap))`.

```ts
import { EPSILON, SECONDS_PER_YEAR } from './constants';
import { applyBgkCorrection } from './bgk';
import { erfc } from './erfc';
import { InvalidSpot, InvalidSigma } from './errors';
import { type Cell, type OracleState, type PricingConfig } from './types';

const SQRT_2 = Math.sqrt(2);

/** Standard normal CDF Φ(x) = ½·erfc(−x/√2). Mirrors `multiplier::normal_cdf`. */
function normalCdf(x: number): number {
  return 0.5 * erfc(-x / SQRT_2);
}

/**
 * Reflection-principle first-passage touch probability for OTM regimes.
 * Mirrors `tap_trading_pricing_engine::first_passage_touch_prob`
 * (multiplier.rs:77-84).
 *   P(touch ≤ τ) = 2·Φ(−|ln(barrier/s₀)| / (σ_sec · √τ))
 */
export function firstPassageTouchProb(
  s0: number, barrier: number, sigmaPerSec: number, tauSec: number,
): number {
  if (sigmaPerSec <= 0 || tauSec <= 0) return 0;
  const b = Math.abs(Math.log(barrier / s0));
  const v = sigmaPerSec * Math.sqrt(tauSec);
  return Math.max(0, Math.min(1, 2 * normalCdf(-b / v)));
}

/**
 * Mirror of `tap_trading_pricing_engine::compute_p_touch` (multiplier.rs:22-66).
 *
 * Validates oracle inputs (throws InvalidSpot / InvalidSigma — spec §7.1
 * mandates the caller pauses taps), returns 0 if the cell already closed,
 * applies BGK correction, then dispatches by spot position:
 *   - S₀ ∈ (L_corr, H_corr) → 1.0 (in-band)
 *   - σ_per_sec == 0        → 0.0 (cannot move)
 *   - S₀ ≤ L_corr           → first_passage to L_corr from below
 *   - S₀ ≥ H_corr           → first_passage to H_corr from above
 *
 * Does NOT call hui_no_touch — that's a separate library function for
 * offline double-barrier valuation and is not in the runtime path.
 */
export function computePTouch(
  cell: Cell, oracle: OracleState, cfg: PricingConfig, nowMs: number,
): number {
  if (!(Number.isFinite(oracle.spot) && oracle.spot > 0)) {
    throw new InvalidSpot(oracle.spot);
  }
  if (!(Number.isFinite(oracle.sigma_annualized) && oracle.sigma_annualized >= 0)) {
    throw new InvalidSigma(oracle.sigma_annualized);
  }
  if (cell.t_close_ms <= nowMs) return 0;

  const tauCloseSec = (cell.t_close_ms - nowMs) / 1000;
  const sigmaPerSec = (oracle.sigma_annualized / Math.sqrt(SECONDS_PER_YEAR)) * cfg.jump_buffer;
  const m = tauCloseSec / cfg.tick_period_seconds; // matches Rust: τ / 0.05
  const [lCorr, hCorr] = applyBgkCorrection(
    cell.strike_lo, cell.strike_hi, sigmaPerSec, tauCloseSec, m,
  );

  let pTouch: number;
  if (oracle.spot > lCorr && oracle.spot < hCorr) {
    pTouch = 1;
  } else if (sigmaPerSec === 0) {
    // OTM with zero vol: spot cannot move, so cannot touch.
    pTouch = 0;
  } else if (oracle.spot <= lCorr) {
    pTouch = firstPassageTouchProb(oracle.spot, lCorr, sigmaPerSec, tauCloseSec);
  } else {
    pTouch = firstPassageTouchProb(oracle.spot, hCorr, sigmaPerSec, tauCloseSec);
  }
  return Math.max(0, Math.min(1, pTouch));
}

/**
 * Mirror of `tap_trading_pricing_engine::compute_multiplier` (multiplier.rs:95-115).
 *
 * Order of operations matches Rust exactly:
 *   1. closed cell (t_close <= now) → 0.0   (does NOT validate oracle)
 *   2. compute τ, floor
 *   3. computePTouch validates spot/sigma — throws InvalidSpot / InvalidSigma
 *   4. raw = p_touch < EPSILON ? cap : (1 - house_margin) / p_touch
 *   5. clamp into [floor, cap]
 */
export function computeMultiplier(
  cell: Cell, oracle: OracleState, cfg: PricingConfig, nowMs: number,
): number {
  if (cell.t_close_ms <= nowMs) return 0;

  const tauCloseSec = (cell.t_close_ms - nowMs) / 1000;
  const floor = cfg.floor_a + cfg.floor_b * tauCloseSec;

  const pTouch = computePTouch(cell, oracle, cfg, nowMs);
  const raw = pTouch < EPSILON
    ? cfg.multiplier_cap
    : (1 - cfg.house_margin) / pTouch;

  return Math.min(cfg.multiplier_cap, Math.max(floor, raw));
}
```

```rust
//! Run: cargo run --bin gen_fixtures > ../../ui/tests/fixtures/parity.json

//! - "ok"            → Rust returned Ok(m); TS must match within 1e-6
//! - "boundary_zero" → Rust returned Ok(0.0) (closed cell); TS must match
//! - "invalid_input" → Rust returned Err(InvalidSpot|InvalidSigma); TS must throw the matching error class
//! - "convergence_failure" → Rust returned Err(HuiConvergenceFailure); TS must throw

use serde::Serialize;
use tap_trading_pricing_engine::{
    compute_multiplier, AssetSymbol, Cell, OracleState, PricingConfig, PricingError,

#[derive(Serialize)]

fn case(s0: f64, lo: f64, hi: f64, sigma: f64, tau_sec: f64) -> Case {
    let now_ms = 1_000_000i64;
    let t_close_ms = now_ms + (tau_sec * 1000.0) as i64;
    let cell = Cell {
        asset: AssetSymbol::Eth,
        strike_lo: lo, strike_hi: hi,
        t_open_ms: now_ms as u64,
        t_close_ms: if tau_sec > 0.0 { t_close_ms as u64 } else { now_ms as u64 },
    };

        asset: AssetSymbol::Eth,
        spot: s0,
        sigma_annualized: sigma,
```

```ts
export const CELL_DURATION_MS = 5_000;
export const ETH_STRIKE_STEP = 0.5;

export function nextCellOpenMs(nowMs: number): number {
  return Math.floor(nowMs / CELL_DURATION_MS + 1) * CELL_DURATION_MS;
}

/**
 * Returns `rows` strike levels centered around the current price,
 * snapped to the global ETH ladder (0.5-step from 0).
 */
export function strikeLadderEth(currentMid: number, rows: number): number[] {
  const center = Math.round(currentMid / ETH_STRIKE_STEP) * ETH_STRIKE_STEP;
  const half = Math.floor(rows / 2);
  const out: number[] = [];
  for (let i = -half; i < rows - half; i++) {
    out.push(+(center + i * ETH_STRIKE_STEP).toFixed(2));
  }
  return out;
}

export function cellKey(strikeLo: number, strikeHi: number, tOpenMs: number): string {
  return `${strikeLo.toFixed(2)}-${strikeHi.toFixed(2)}-${tOpenMs}`;
}
```

```ts
import { useEffect, useState } from 'react';
import { CELL_DURATION_MS, ETH_STRIKE_STEP, cellKey, nextCellOpenMs, strikeLadderEth } from '@/lib/time';
import type { Cell } from '@/pricing/types';

const ROWS = 12;
const COLS_MOBILE = 4;
const COLS_DESKTOP = 6;

interface VisibleGrid {
  cells: Cell[][];   // cells[col][row] — outer = time columns, inner = strike rows
  keys: string[][];
  ladder: number[];
  cols: number;
  rows: number;
  nowMs: number;
}

export function useVisibleCells(currentMid: number | null): VisibleGrid {
  const [now, setNow] = useState(() => Date.now());
  useEffect(() => {
    const id = setInterval(() => setNow(Date.now()), 250);
    return () => clearInterval(id);
  }, []);
  const cols = typeof window !== 'undefined' && window.innerWidth >= 1024 ? COLS_DESKTOP : COLS_MOBILE;
  const ladder = currentMid !== null ? strikeLadderEth(currentMid, ROWS) : Array(ROWS).fill(0);
  const firstOpen = nextCellOpenMs(now);

  const cells: Cell[][] = [];
  const keys: string[][] = [];
  for (let c = 0; c < cols; c++) {
    const tOpen = firstOpen + c * CELL_DURATION_MS;
    const tClose = tOpen + CELL_DURATION_MS;
    const colCells: Cell[] = [];
    const colKeys: string[] = [];
    for (let r = 0; r < ROWS; r++) {
      const strikeLo = ladder[r];
      const strikeHi = ladder[r] + ETH_STRIKE_STEP;
      colCells.push({ asset: 'ETH', strike_lo: strikeLo, strike_hi: strikeHi, t_open_ms: tOpen, t_close_ms: tClose });
      colKeys.push(cellKey(strikeLo, strikeHi, tOpen));
    }
    cells.push(colCells);
    keys.push(colKeys);
  }
  return { cells, keys, ladder, cols, rows: ROWS, nowMs: now };
}
```

Two correctness notes:

- The aggregator emits ticks keyed by `(asset, run_id, seq)`; a new `run_id` (aggregator restart) means the series has restarted and σ must reseed. Drop the log-return across a `run_id` boundary.
- `vol.ts` now throws on bad inputs. Guard against `Math.log(non-positive)` (produces NaN) by skipping the update; a NaN log return would otherwise throw `InvalidLogReturn`.

```ts
// ...
interface Snapshot {
  tick: OracleTick | null;
  wsState: WsState;
  wsLastEventMs: number;
  vol: VolState;
}

const FRESH_VOL: VolState = INITIAL_VOL_STATE;

let snapshot: Snapshot = {
  tick: null,
  wsState: 'closed',
  wsLastEventMs: 0,
  vol: FRESH_VOL,
};
// ...
export const tickStore = {
  // ...
  push(tick: OracleTick) {
    const prev = snapshot.tick;
    // Reset σ across a run_id boundary (aggregator restart) or asset switch.
    const sameStream = prev !== null && prev.asset === tick.asset && prev.run_id === tick.run_id;
    let vol = sameStream ? snapshot.vol : FRESH_VOL;
    if (sameStream && prev.mid > 0 && tick.mid > 0) {
      const logReturn = Math.log(tick.mid / prev.mid);
      if (Number.isFinite(logReturn)) {
        vol = nextVol(vol, logReturn, 0.94);
      }
    }
    snapshot = { tick, wsState: 'open', wsLastEventMs: Date.now(), vol };
    emit();
  },
  reset() {
    snapshot = { ...snapshot, vol: FRESH_VOL, tick: null };
    emit();
  },
};
```

## Task 15: `useCellMultiplier` — cells show live μ

**Files:**
- Create: `games/tap-trading/ui/src/hooks/useCellMultiplier.ts`
- Modify: `games/tap-trading/ui/src/components/Cell.tsx`

- [ ] **Step 1: Create the hook**

Perf note. The spec budget is 240 computes/sec (10 Hz × 24 cells); naively recomputing inside `getSnapshot` on every tick fanout (20 Hz × 48–72 cells) runs at 4–6× that. We bound the rate two ways:

1. Quantize `nowMs` to a 100 ms bucket (10 Hz) so consecutive ticks in the same bucket reuse the cached value.
2. Cache the last `(cellKey, mid, sigma, bucketMs) → multiplier` per cell in a module-scoped Map. `getSnapshot` reads or computes, but never returns a *fresh* number when nothing changed — that satisfies `useSyncExternalStore`'s referential-stability requirement, not just value equality.

```ts
import { useSyncExternalStore } from 'react';
import { tickStore } from '@/lib/tick-store';
import { computeMultiplier } from '@/pricing/multiplier';
import { InvalidSpot, InvalidSigma } from '@/pricing/errors';
import { HuiConvergenceFailure } from '@/pricing/hui';
import { DEFAULT_PRICING_CONFIG, type Cell } from '@/pricing/types';
import { cellKey } from '@/lib/time';

interface CacheEntry { key: string; mult: number | null; }
const CACHE_CAP = 256;        // bound: ~5–6× the active visible-grid size
const cache = new Map<string, CacheEntry>();

function bucket(nowMs: number): number {
  return Math.floor(nowMs / 100) * 100;
}

function cachePut(ck: string, entry: CacheEntry) {
  // LRU-ish via insertion order: re-key on update so the entry moves to the tail.
  if (cache.has(ck)) cache.delete(ck);
  cache.set(ck, entry);
  if (cache.size > CACHE_CAP) {
    const oldest = cache.keys().next().value;
    if (oldest !== undefined) cache.delete(oldest);
  }
}

export function useCellMultiplier(cell: Cell, nowMs: number): number | null {
  const ck = cellKey(cell.strike_lo, cell.strike_hi, cell.t_open_ms);
  return useSyncExternalStore(
    tickStore.subscribe,
    () => {
      const snap = tickStore.getSnapshot();
      // Spec §"Error taxonomy": WS drop / no tick → cells dim. Do NOT return a
      // stale cached μ; render `—` so the cell is visibly untappable.
      if (!snap.tick || !Number.isFinite(snap.tick.mid)) return null;

      const inputKey = `${ck}|${snap.tick.mid}|${snap.vol.sigmaAnnualized}|${bucket(nowMs)}`;
      const cached = cache.get(ck);
      if (cached && cached.key === inputKey) return cached.mult;

      let mult: number | null;
      try {
        const oracle = {
          asset: 'ETH' as const,
          spot: snap.tick.mid,
          sigma_annualized: snap.vol.sigmaAnnualized,
          timestamp_ms: snap.tick.ts_ms,
        };
        const m = computeMultiplier(cell, oracle, DEFAULT_PRICING_CONFIG, nowMs);
        // Snap to 2 decimals — UI never needs more, and bucketing already caps
        // the recompute rate.
        mult = m > 0 ? Math.round(m * 100) / 100 : null;
      } catch (e) {
        if (e instanceof InvalidSpot || e instanceof InvalidSigma || e instanceof HuiConvergenceFailure) {
          mult = null; // render `—`, untappable until the next good tick
        } else {
          throw e;
        }
      }
      cachePut(ck, { key: inputKey, mult });
      return mult;
    },
  );
}
```

```tsx
  ladder: number[];          // strike values (rows top→bottom or bottom→top?)
  cellHeightPx: number;
  cellWidthPx: number;
  rows: number;
  cols: number;
  nowMs: number;

  const { ladder, cellHeightPx, cellWidthPx, rows, cols, nowMs } = props;

  const width = cols * cellWidthPx;
  const height = rows * cellHeightPx;

  return (
    <svg className="pointer-events-none absolute inset-0" width={width} height={height}>
      <polyline points={points} fill="none" stroke="#00B8FF" strokeWidth="2" opacity="0.7" />
```

```ts
export type LifecycleState = 'PENDING' | 'LOCKED' | 'WON' | 'LOST' | 'VOIDED' | 'REJECTED';

export interface CellPosition {
  cellKey: string;
  clientRequestId: string;
  positionId?: number;
  stake: number;
  state: LifecycleState;
  multiplierAtTap?: number;
  rejectReason?: string;
  settledAtMs?: number;
  tCloseMs: number;
}

const positions = new Map<string, CellPosition>();
const listeners = new Set<() => void>();
function emit() { listeners.forEach((cb) => cb()); }

export const positionsStore = {
  subscribe(cb: () => void) { listeners.add(cb); return () => listeners.delete(cb); },
  getSnapshot(): ReadonlyMap<string, CellPosition> { return positions; },
  get(cellKey: string): CellPosition | undefined { return positions.get(cellKey); },
  set(cellKey: string, p: CellPosition) { positions.set(cellKey, p); emit(); },
  update(cellKey: string, patch: Partial<CellPosition>) {
    const cur = positions.get(cellKey);
    if (!cur) return;
    positions.set(cellKey, { ...cur, ...patch });
    emit();
  },
  delete(cellKey: string) { positions.delete(cellKey); emit(); },
  pendingStake(): number {
    let s = 0;
    positions.forEach((p) => { if (p.state === 'PENDING') s += p.stake; });
    return s;
  },
};
```

- [ ] **Step 2: Implement `useTap`**

`src/hooks/useTap.ts`. Imports are static (the dynamic-import dance on the hot path adds an unpredictable chunk-fetch round trip on every fresh entry). The server's `TapResponse.status` is a free-form `String` that may be any of `OPEN | WON | LOST | VOIDED` — the API hands back the *current* persisted status, which for a fresh 201 is always `OPEN` but for an idempotent 200-replay can be any settled state.

```ts
import { useMutation, useQueryClient } from '@tanstack/react-query';
import { api, ApiError } from '@/lib/api';
import { positionsStore, type LifecycleState } from '@/lib/positions-store';
import { cellKey as makeCellKey } from '@/lib/time';
import { tickStore } from '@/lib/tick-store';
import { computeMultiplier } from '@/pricing/multiplier';
import { DEFAULT_PRICING_CONFIG, type Cell } from '@/pricing/types';
import { InvalidSpot, InvalidSigma } from '@/pricing/errors';
import { HuiConvergenceFailure } from '@/pricing/hui';

const STAKE_V0 = 100;

interface TapResponse {
  position_id: number;
  multiplier_at_tap: number;
  status: string; // server-side String; we normalize via mapStatus()
  t_close_ms: number;
}

function mapStatus(s: string): LifecycleState {
  switch (s) {
    case 'OPEN':   return 'LOCKED';
    case 'WON':    return 'WON';
    case 'LOST':   return 'LOST';
    case 'VOIDED': return 'VOIDED';
    default:       return 'LOCKED';
  }
}

export function useTap() {
  const qc = useQueryClient();

  return useMutation({
    mutationFn: async (cell: Cell) => {
      const key = makeCellKey(cell.strike_lo, cell.strike_hi, cell.t_open_ms);
      if (positionsStore.get(key)) {
        throw new Error('cell already tapped');
      }
      const snap = tickStore.getSnapshot();
      if (!snap.tick) throw new Error('no tick yet');
      if (Date.now() + 1000 >= cell.t_close_ms) throw new Error('lock window');

      const clientRequestId = crypto.randomUUID();
      positionsStore.set(key, {
        cellKey: key,
        clientRequestId,
        stake: STAKE_V0,
        state: 'PENDING',
        tCloseMs: cell.t_close_ms,
      });

      // Client multiplier for the drift check. The server recomputes and uses
      // its own value; this is only consulted by the 3% gate.
      let clientMultiplier: number;
      try {
        clientMultiplier = computeMultiplier(
          cell,
          {
            asset: 'ETH',
            spot: snap.tick.mid,
            sigma_annualized: snap.vol.sigmaAnnualized,
            timestamp_ms: snap.tick.ts_ms,
          },
          DEFAULT_PRICING_CONFIG,
          Date.now(),
        );
      } catch (e) {
        // Pricing-engine errors mean we can't tap right now (bad oracle data,
        // wide-band non-convergence). Mark REJECTED and bubble — toast will fire.
        if (e instanceof InvalidSpot || e instanceof InvalidSigma || e instanceof HuiConvergenceFailure) {
          positionsStore.update(key, { state: 'REJECTED', rejectReason: 'client_pricing' });
          setTimeout(() => positionsStore.delete(key), 800);
          throw e;
        }
        throw e;
      }

      try {
        const resp = await api<TapResponse>('/v1/positions', {
          method: 'POST',
          body: {
            client_request_id: clientRequestId,
            asset: 'ETH',
            strike_lo: cell.strike_lo,
            strike_hi: cell.strike_hi,
            t_open_ms: cell.t_open_ms,
            t_close_ms: cell.t_close_ms,
            stake_points: STAKE_V0,
            client_multiplier: clientMultiplier,
            oracle_seq_at_tap: snap.tick.seq,
            oracle_run_id_at_tap: snap.tick.run_id,
          },
        });
        positionsStore.update(key, {
          state: mapStatus(resp.status),
          positionId: resp.position_id,
          multiplierAtTap: resp.multiplier_at_tap,
        });
        qc.invalidateQueries({ queryKey: ['me'] });
        return resp;
      } catch (err) {
        const reason = err instanceof ApiError ? err.code : 'network';
        positionsStore.update(key, { state: 'REJECTED', rejectReason: reason });
        // Brief flash, then delete so cell becomes tappable again.
        setTimeout(() => positionsStore.delete(key), 800);
        throw err;
      }
    },
  });
}
```

- [ ] **Step 1: Implement `usePositionPoll`**

`PositionResponse` (per `api/src/handlers/positions.rs` `get_position_by_id`) has the shape `{ id, asset, status, t_open_ms, t_close_ms, stake_points, multiplier_at_tap }`. There is **no** `settled_at_ms` on this endpoint — the settlement timestamp is exposed only on `/v1/me/history` inside the nested `settlement` object. For the local store marker we use `Date.now()`.

```ts
import { useEffect } from 'react';
import { useQuery, useQueryClient } from '@tanstack/react-query';
import { api } from '@/lib/api';
import { positionsStore } from '@/lib/positions-store';

interface PositionResponse {
  id: number;
  asset: string;
  status: 'OPEN' | 'WON' | 'LOST' | 'VOIDED';
  t_open_ms: number;
  t_close_ms: number;
  stake_points: number;
  multiplier_at_tap: number;
}

export function usePositionPoll(positionId: number | undefined, cellKey: string, tCloseMs: number) {
  const qc = useQueryClient();
  const q = useQuery({
    queryKey: ['position', positionId],
    enabled: positionId !== undefined,
    queryFn: () => api<PositionResponse>(`/v1/positions/${positionId}`),
    refetchInterval: (query) => {
      const data = query.state.data;
      if (data && data.status !== 'OPEN') return false;
      if (Date.now() > tCloseMs + 10_000) return false;
      return 500;
    },
  });

  useEffect(() => {
    if (!q.data) return;
    if (q.data.status === 'OPEN') return;
    const state = q.data.status === 'WON' ? 'WON' : q.data.status === 'LOST' ? 'LOST' : 'VOIDED';
    positionsStore.update(cellKey, { state, settledAtMs: Date.now() });
    qc.invalidateQueries({ queryKey: ['me'] });
    qc.invalidateQueries({ queryKey: ['me', 'history'] });
  }, [q.data, cellKey, qc]);
}
```

Wire-ups: `useTap` calls `telemetry.tap` before POST, `telemetry.locked` on 201 success, `telemetry.drift` on `drift_exceeded`, `telemetry.reject` for every other rejection. `usePositionPoll` calls `telemetry.settled` when status flips. `ws.ts` calls `telemetry.wsDrop` in the close handler.

- [ ] **Step 2: Map error codes to toast messages in `useTap`**

The full set of codes the API emits (per `api/src/error.rs`) is:
`missing_account_id`, `invalid_account_id`, `invalid_stake`, `unknown_asset`,
`lock_window`, `invalid_cell`, `stale_quote`, `drift_exceeded`,
`insufficient_balance`, `rate_limited`, `not_found`, `forbidden`, `internal`.
On `drift_exceeded` the body carries `server_multiplier`. On `rate_limited`
the response carries a `Retry-After` header.

In the catch block of `useTap.ts`:

```ts
import { toast } from '@/lib/toast';
import { telemetry } from '@/lib/telemetry';
import { ApiError } from '@/lib/api';
// ...
} catch (err) {
  const reason = err instanceof ApiError ? err.code : 'network';
  // Drift is the canary for TS-port bugs — log both values so we can audit.
  if (err instanceof ApiError && reason === 'drift_exceeded') {
    telemetry.drift(err.serverMultiplier ?? NaN, clientMultiplier);
  } else {
    telemetry.reject(key, reason);
  }
  const msg = errToast(err, reason);
  toast.warn(msg);
  positionsStore.update(key, { state: 'REJECTED', rejectReason: reason });
  setTimeout(() => positionsStore.delete(key), 800);
  throw err;
}

function errToast(err: unknown, reason: string): string {
  if (err instanceof ApiError) {
    switch (reason) {
      case 'stale_quote':         return 'quote moved, try again';
      case 'drift_exceeded':      return `price drifted (server ${err.serverMultiplier?.toFixed(2) ?? '—'}×)`;
      case 'insufficient_balance':return 'out of points';
      case 'lock_window':         return 'too late, cell closing';
      case 'invalid_cell':
      case 'invalid_stake':
      case 'unknown_asset':       return 'invalid request (bug)';
      case 'missing_account_id':
      case 'invalid_account_id':  return 'session lost, reload';
      case 'forbidden':           return 'not your position';
      case 'rate_limited':        return `slow down (retry in ${err.retryAfterSecs ?? 1}s)`;
      case 'not_found':           return 'position not found';
      case 'internal':            return 'server hiccup';
    }
    if (err.status === 429) return 'slow down (retry in 1s)';
    if (err.status >= 500)  return 'server hiccup';
  }
  return 'network error';
}
```

```ts
import { positionsStore } from './positions-store';
import { queryClient } from './query-client';

if (typeof document !== 'undefined') {
  document.addEventListener('visibilitychange', () => {
    if (document.visibilityState !== 'visible') return;
    // 1. Force reconnect; the new onopen resets tickStore.
    disconnect();
    connect();
    // 2. Refetch `/v1/me` so balance is current after the tab was idle.
    queryClient.invalidateQueries({ queryKey: ['me'] });
    queryClient.invalidateQueries({ queryKey: ['me', 'history'] });
    // 3 & 4. Re-poke each LOCKED position once; drop stale PENDINGs.
    const ps = positionsStore.getSnapshot();
    ps.forEach((p, k) => {
      if (p.state === 'LOCKED' && p.positionId !== undefined) {
        queryClient.invalidateQueries({ queryKey: ['position', p.positionId] });
      }
      if (p.state === 'PENDING' && Date.now() > p.tCloseMs + 10_000) {
        positionsStore.update(k, { state: 'REJECTED', rejectReason: 'visibility_timeout' });
        setTimeout(() => positionsStore.delete(k), 800);
      }
    });
  });
```
